# CLI Data Maker
  - git clone
  - npm install -g .


# Usage: 
  - cdm -c <sumple_count> -s <sample_name_prefix>

Options:

 > --help              Show help                                        [boolean]
 
 > --version           Show version number                              [boolean]
 
 > -c, --count         Sample count                           [number] [required]
 
 > -s, --sampleprefix  Sample prefix                          [string] [required]
